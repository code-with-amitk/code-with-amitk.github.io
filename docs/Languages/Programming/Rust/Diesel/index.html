What is Diesel

Diesel is a ORM(object-relational mapping).
ORM is programming technique that connects object-oriented programming (OOP) code with relational databases
The foundational thing an ORM does is map data from a database to an object/structs within code (among other things)
Diesel can connect Postgres, MySQL, SQLlite databases
