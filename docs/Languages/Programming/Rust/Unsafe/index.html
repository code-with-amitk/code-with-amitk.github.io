What is Unsafe?

There can be 2 types of Rust.
Safe Rust Unsafe Rust (Like C Programming language)
What Compiler takes care of type-safety, memory-safety, thread safety etc Like C, Compiler does not check safety rules and coder does at its own risk

Why we need Unsafe Rust

We want low level control over implementation details
Maybe you're writing a low-level abstraction not exposed by the standard library
Dereference raw pointers etc
