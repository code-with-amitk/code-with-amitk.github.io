What is Triat in Rust

Interface/class in Rust(declared with keyword trait) having Virtual Functions(not pure Virtual) declared or defined inside it. These functions are implemented(overridden) on type.

Declaration & Definition

Declaration

Trait declared having functions

pub struct Employee{                          //Declared "struct Employee"
    pub name:String,
    pub emp_id:u32,
}

pub struct Contractor{                        //Declared "struct Contractor".
    pub contractor_name:String,
}

pub trait CompanyDB {                         //Trait Declaration
    fn add_to_db(&self);
}

Definition

Trait is implemented for particular types. Eg: add_to_db() function is overridden for Employee, Contractor Type. 

// Trait Definition. Implemented trait/Overridden for Type="Employee struct"                
impl CompanyDB for Employee{                  
    fn add_to_db(&self) {
        println!("Added {} {}", self.name, self.emp_id);
    }
}
