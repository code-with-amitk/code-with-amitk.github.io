What noexcept operator

What is noexcept

throw is deprecated in C++11, will be removed in C++20.
noexcept returns true or false based on calculation that function can throw exception or not.

noexcept operator

noexcept operator returns true if expression is declared to not throw any exceptions.
Even if noexcept(expr) is true, an evaluation of expr may still throw as the result of encountering undefined behavior. 

noexcept(constant_expression)
