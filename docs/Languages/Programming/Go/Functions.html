range()

for iterating over slice or map.
Returns: Iterate over slice(index, element), Iterate over map(key, value)
index or value anything can be skipped using "_" 

for key, value := range hm {        //Map
    out += value
}
for _, val := range(a) {        //Vector
    fmt.Println (val)
}
