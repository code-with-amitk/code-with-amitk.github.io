Companies in RAG Application Space

RAG space can be divided into 6 areas.

Area Companies in that area
Data Storage (vector storage) Pinecone, Zilliz, MongoDB
Data Movement Databricks, Snowflake, Airbyte
Inference First Party: Companies having their own LLMs. eg: openAI, Meta, Anthropic
3rd Party: Serve other LLMs. eg: Predibase, Tensorwave, GMI Cloud
Observability Arize, Comet, Galileo
Orchestration 1st wave: Langchain, LlamaIndex, Langflow
2nd Wave: BAML, Pixeltable
Deployment GCP, Azure, AWS
