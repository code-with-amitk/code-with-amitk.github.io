Calorie prediction (DecisionTreeRegressor)

Calorie prediction using DecisionTreeRegressor Model

Scenario
  You are running a company which need to predict calories needed by atheletes based on duration and intensity(pulse, maxpulse) rates of their workouts
  You need to develop a ML model which takes (Duration,Pulse,Maxpulse,Calories) as input and predicts Calories burned during athlete workout.

Inputs
  You would be provided with data.csv file which contains historical data of atheletes workout sessions 

# data.csv
Duration,Pulse,Maxpulse,Calories
    60,  110,   130,    409.1
    60,  117,   145,    479.0
    60,  103,   135,    340.0
    45,  109,   175,    282.4
Requirement
  You need to write a Machine Learning model which takes (Duration, Pulse, MaxPulse) as input and predict Calories burned by athelete.
  This way they can take appropriate calorie intake during/before their workouts

import pandas as pd                            #1
from sklearn.tree import DecisionTreeRegressor #2

df = pd.read_csv("data.csv")                   #3
print("Raw data:\n", df.to_string())

# Clean the data, remove empty rows
clean_df = df.dropna(axis=0)                   #4
print("\nCleaned data:\n", clean_df.to_string())

print("\n########### Traning Begin ###########")
X_df_training_data = clean_df.dropna(axis=0)[['Duration','Pulse','Maxpulse']] #5
print("Traning Data(Removed Calories):\n", X_df_training_data.to_string())

# Predict Calories, Remove calories from training data                  #6
y_predict_calories = clean_df['Calories']
print("(Label)Calories. For above data, these should be calories:\n", 
        y_predict_calories.to_string())

model = DecisionTreeRegressor(random_state=1)                         #7
model.fit(X_df_training_data, y_predict_calories)                     #8
calories_prediction_on_traningD = model.predict(X_df_training_data)   #9

# This is overfitting
print("Model Predicted: Calories:", calories_prediction_on_traningD)
print("########### Traning End ###########")

print("\n########### Real World ###########")
#Real-world data                                                     #10
real_data = pd.DataFrame({
    'Duration':[55, 45, 60],
    'Pulse':[120, 125, 117],
    'Maxpulse':[150, 165, 145]
})
print("Real World Data:\n", real_data)

pred = model.predict(real_data)                                     #11
print("Calories Predictions: ", pred)
