Ollama (Inference Engine)

Ollama is a model runner (also called an inference engine). Ollama runs the models like: Llama, Gemma, Phi etc

Ollama Installation & Run


$ wget https://ollama.com/download/ollama-linux-amd64.tar.zst
$ tar -xvf ollama-linux-amd64.tar.zst
it creates 2 directories bin, lib

$ mv bin/ollama /usr/bin

Model / AI Model

What is AI Model?
This is not an executable(exe), or shared library(so file). It doesn’t contain machine instructions (x86_64 or ARM), and your operating system cannot run it directly
An AI model is a massive multidimensional array of numbers (a static data file), coupled with a specific file structure metadata layout. These numbers are called weights and biases.
When a model "learns" during training, it isn't writing code. It is adjusting millions or billions of floating-point numbers until those numbers mathematically represent patterns in data.
Examples:
- Opensource Models: Llama, Gemma, Phi, Llama 3.2
- Closedsource Models: gpt(openai), Claude (Anthropic).

Model Efficiency?

1. Parameters: The "1B", "3B", or "8B" stands for Billion Parameters(the number of internal connections inside the AI's brain). More parameters usually mean a smarter model, but it requires more memory.
2. Quantization: Floating point numbers for weights/calculations.
3. Speed (Tokens per Second): Tokens are fractions of words (100 tokens $\approx$ 75 words).An efficient model running on a good GPU will hit 40 to 100+ tokens per second, printing text faster than you can read it. If it drops below 10-15 tokens/sec, it feels sluggish.
4. Resource Utilization(Fitting in VRAM): Does the model fit entirely inside your Graphics Card's memory (VRAM)? If it fits in VRAM, it runs at lightning speed. If it overflows into your system RAM, performance drops off a cliff.

Can Ollama run any model?

Ollama understands GGUF format. Any model which can be converted to this can run on Ollama
1. Direct Support: Ollama has an official library supporting dozens of mainstream open models (Llama 3, Gemma 2, Mistral, Qwen, DeepSeek). You can download these instantly with ollama run
2. Custom Models: If you find a niche or custom-tuned open-source model on Hugging Face (the GitHub of AI), you can download its .gguf file and use a simple file called a Modelfile to load it into Ollama manually.
