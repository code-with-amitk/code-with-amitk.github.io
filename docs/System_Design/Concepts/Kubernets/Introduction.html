Kubernets?

Kubernets(Container Orchestrator)

Orhestration means managing the lifecycles of containers. These are GUI/tools to automate following tasks: Deployment, Scaling up/down, Movement of containers from one host to another, Load balancing, Health monitoring.
Examples of Container Orhestrators: Kubernets, Docker swarm, Nomad Kubernets Cluster has different microservices inside it.
