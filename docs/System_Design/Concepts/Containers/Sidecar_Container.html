Sidecar Container

A sidecar is a helper container that runs alongside your main application container in the same task/pod. It extends or enhances the application's functionality without modifying the application itself.

Some Usecases of Sidecar container
1. Logging for main container
2. SSL Encryption/Decryption for traffic to main container

Main vs Sidecar

Main container Sidecar
Role Runs core application Provides support functionality
Lifetime Ties to main application Ties to main application
