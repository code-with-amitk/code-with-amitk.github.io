What is Load Balancer?

Node which evenly distributes incoming traffic among web servers that are defined in a load-balanced set.
Users connect to the public IP of the load balancer, and web-servers are reachable over pvt IP from load balancer.

LB Solves Problem = FAILOVER

If server 1 goes offline, all the traffic will be routed to server 2. This prevents the website from going offline. We will also add a new healthy web server to the server pool to balance the load.
If the website traffic grows rapidly, and two servers are not enough to handle the traffic, the load balancer can handle this problem gracefully.
