What is VPC

VPC (Virtual Pvt Cloud)

This is a private, isolated virtual network environment where you can launch AWS resources(eg EC2) into subnets that you define
It allows you to provision a logically isolated section of the AWS Cloud
Within your VPC, you can define your own IP address range subnets, route tables, and network gateways as you see fit
