    Problem:

    How to implement transaction that span multiple microservices? ie Distributed Transaction. 
    
Microservice1 --txn1_1---\
                          \
Microservice3 --txn1_2--- DB
    Distributed Transactions would not be ACID but only ACD
      Microservice1 writes to DB using txn1_1 and microservice2 writes to DB using txn1_2. These are 2 seperate transactions carried independently
      But they are part of only 1 transaction, ie the distributed transaction
      Since txn1_2 might change something in txn1_1, to create 1 transaction. These cannot be isolated
