Methods of IPC

Method Communication Synchronization
1. Pipe, Rust Channels Parent to child viceversa Yes
2.Named pipes, fifo Multiple process can communicate. Process-1 to process-2
3.Message Queues Multiprocess. Process-1 to process-2
Rust: v_queue, posixmq, Channels are also sort of Message Queues 		Yes
4.Shared Memory Multiprocess. Process-1 to process-2. Fastest No
5.Memory mapped files Map a file onto process Virtual address space and do RW.
Advantages? 1. Mapping a file is cheaper than using I/O system calls for RW 2. Same file can be mapped to 2-or-more processes at same time. Write by 1 is visible to other instantly.
