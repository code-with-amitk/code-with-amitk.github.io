User Space vs Kernel Space Threads

User Space threads Kernel Space threads
What A user-space thread is any thread whose scheduling is primarily handled in user space instead of Kernel Space. Kernel does not know these threads exist.
The runtime may schedule User threads over:
- one kernel thread (M:1)
- several kernel threads (M:N)
- or even dedicate one kernel thread per task (less common).
