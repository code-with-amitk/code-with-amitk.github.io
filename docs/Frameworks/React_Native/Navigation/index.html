Navigation Creating 2 screens

Navigation

Managing the presentation of, and transition between, multiple screens is typically handled by what is known as a navigator. 

$ npm install @react-navigation/native @react-navigation/native-stack

// Install for expo managed project
$ npx expo install react-native-screens react-native-safe-area-context

Creating 2 screens

We will create 2 screens (home, profile) and navigate between them 

import {Text, Button} from 'react-native';
import * as React from 'react';
import {NavigationContainer} from '@react-navigation/native';
import {createNativeStackNavigator} from '@react-navigation/native-stack'   //1

const HomeScreen = ({navigation}: any) => {                     //5
    return (
    <Button
        title="Go to Jane's profile"
        onPress={() =>
        navigation.navigate('Profile', {name: 'Jane'})          //5
        }
    />
    );
};
const ProfileScreen = ({navigation, route}: any) => {
    return <Text>This is {route.params.name}'s profile</Text>;
};

const Stack = createNativeStackNavigator();
const Main = () => {
    return (
        <Stack.Navigator>            //2
            <Stack.Screen name="Home" component={HomeScreen} options={{title: 'Welcome Screen'}} />      //3, 4
            <Stack.Screen name="Profile" component={ProfileScreen} options={{title: 'Profile Screen'}}/>
        </Stack.Navigator>
    );
};

export default Main;
1. import createNativeStackNavigator
2. Whole app need to wrapped inside createNativeStackNavigator().Navigator in App.js
3. Stack.Screen defines 2 screens:
  name="Home"
  name="Profile"
4. We can set options such as the screen title for each screen in the options prop of Stack.Screen.
options={{title: 'Welcome Screen'}}
options={{title: 'Profile Screen'}}
5. Each screen takes navigation prop. This prop has various methods to link to other screen
  navigation.navigate to go to profile screen.
