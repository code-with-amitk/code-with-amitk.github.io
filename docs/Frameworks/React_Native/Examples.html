Expo Getting Started
npx(Node Package execute) is NPM package runner. It allows developers to execute any Javascript Package available on the NPM registry
npm(Node Package Manager) a free, open-source registry and command-line tool for JavaScript software packages 

// Open cmd prompt on Windows, and move to C:

C:\>mkdir react

C:\>cd react

C:\react>npx create-expo-app@latest
Need to install the following packages:
create-expo-app@3.5.3
Ok to proceed? (y) y

C:\react>cd my-app

C:\react\my-app>npx expo start --clear
Starting project at C:\react\my-app
React Compiler enabled
Starting Metro Bundler
warning: Bundler cache is empty, rebuilding (this may take a minute)
