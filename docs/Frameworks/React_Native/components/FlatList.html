What is FlatList?

displays a scrolling list of changing, but similarly structured, data. Unlike the more generic ScrollView, the FlatList only renders elements that are currently showing on the screen, not all the elements at once.
Designed to efficiently render large lists of data with optimal performance. FlatList component requires two props: data and renderItem

Examples

Simple FlatList


import {View, FlatList, Text, Image, TextInput} from 'react-native';    //1

const data = [                                  //2 
    {id:1, cust:"alex", ph:"123", follow:"1/2/2025"},
    {id:2, cust:"bob", ph:"345", follow:"1/2/2025"}
];

interface DataItem {                            //3
    id: number;
    cust: string;
    ph: string;
    follow: string;
}
const FlatListComp = () => {
    const renderItem = ({ item }: { item: DataItem }) => (  
    <View>
        <Text>Customer: {item.cust}</Text>
        <Text>Phone: {item.ph}</Text>
        <Text>Follow-up: {item.follow}</Text>
    </View>
    );

    return (
    <FlatList                                    //4
        data={data}
        renderItem={renderItem}
        keyExtractor={(item) => item.id.toString()}
    />
    );
};

const Main = () => {
    return (
        <FlatListComp />
    );
};

export default Main;
1. import
2. Create Array of Object in JavaScript. [] is Array, {} is Object

  Array/list/collection [] is collection of values, Can contain multiple elements of different types
  Object/HashMap/Dictionary {}: key-value pair collection
  Nested Objects in Arrays. Placing {} inside [] creates an array of objects

3. Interface: Defines the shape of data

4. FlatList props

<FlatList
    data={data}                     // The array of items to render
    renderItem={renderItem}         // Function defining how each item is displayed
    keyExtractor={(item) => item.id.toString()}  // Unique key generator
/>
This format will remain consistent across React Native FlatList implementations:
  data provides the source array
  renderItem defines rendering logic
  keyExtractor generates unique keys for list items
