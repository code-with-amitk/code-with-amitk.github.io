What DLPaaS

DLP (Data Loss Prevention)

if sensitive data is leaked(on cloud), then DLP protects the data by inspecting the cloud services.
When user voilates a DLP Policy(in netskope) a log is generated

DLP As a Service (DLPaaS)

Create async service on AWS cloud, listening on REST APIs, where customer can send data for DLP inspection
Service will inspect the data and return results.
