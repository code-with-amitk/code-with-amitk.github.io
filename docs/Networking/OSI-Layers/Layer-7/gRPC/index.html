What is gRPC?

Google Remote Procedure Calls open-source framework for building APIs(similar to REST API).
It uses HTTP/2 for transport and Protocol Buffers (Protobuf) for efficient data serialization
Remote Procedure Call (RPC): A communication model where a client requests a function to run on a remote server as if it were local.
Protocol Buffers (Protobuf): Defines data structures and service interfaces in .proto files, then compiles them into code for various languages, ensuring fast, compact data exchange.
