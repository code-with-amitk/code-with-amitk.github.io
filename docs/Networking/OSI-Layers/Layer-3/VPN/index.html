VPN (Virtual Private Network)

VPN creates secure/encrypted tunnels over less secure network like internet.
Functions of a VPN
1. Data Encryption: MIM, ISP, or network administrators cannot see what you are browsing.
2. IP Masking: Adds a extra header over actual IP header, routers inbetween cannot see for whom packet is destinated
3. Bypassing Geo-Restrictions: It lets you connect to servers in other countries, allowing you to bypass regional content blocks.
