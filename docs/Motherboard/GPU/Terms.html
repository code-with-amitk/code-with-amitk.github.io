How Oracle VirtualBox's provides 1 GPU to all guest VMs

Hardware 3D drivers are installed in guest OS. VirtualBox adds some Windows OS system files to enable Direct3D acceleration. Tunnels 3D commands from the guest to the host GPU using a special driver and communication channel
